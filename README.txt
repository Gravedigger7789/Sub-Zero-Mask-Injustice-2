# Summary

I added thickness (solidify modifier) to the model so when you slice it you don't only end up with only floating tubes. This process also made the tubes hollow.

I included the low poly model with no subsurf modifications and the medium and high poly model with the Catmull subsurf modifier applied. If you'd like to use a different modifier feel free to do so with the attached blender file.

I have not printed this yet but I do plan on doing so. Once I print it I will update the model with the proper scale and sliced pieces.

The source's original model comes from the model posted by OGLoc069: https://ogloc069.deviantart.com/art/Injustice-2-Sub-Zero-733074209
